    private static _convertPNGtoDIB(src: Buffer, width: Number, height: Number, bpp: Number): Buffer source

    Convert a PNG of the byte array to the DIB (Device Independent Bitmap) format.

    PNG in color RGBA (and more), the coordinate structure is the Top/Left to Bottom/Right. DIB in color BGRA, the coordinate structure is the Bottom/Left to Top/Right.

    Params:

    NameTypeAttributeDescription
    src Buffer

    Target image.
    width Number

    The width of the image.
    height Number

    The height of the image.
    bpp Number

    The bit per pixel of the image.

    Return:

    Buffer

    Converted image
