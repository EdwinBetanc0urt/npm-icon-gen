    public ICODirectory: Object source

    Directory information of the ICO file.

    Properties:

    NameTypeAttributeDescription
    width Number

    1 bytes. Specifies image width in pixels. Can be any number between 0 and 255. Value 0 means image width is 256 pixels.
    height Number

    1 bytes. Specifies image height in pixels. Can be any number between 0 and 255. Value 0 means image height is 256 pixels.
    colors Number

    1 bytes. Specifies number of colors in the color palette. Should be 0 if the image does not use a color palette.
    reserved Number

    1 bytes. Reserved. Should be 0.
    planes Number

    2 bytes. In ICO format: Specifies color planes. Should be 0 or 1. In CUR format: Specifies the horizontal coordinates of the hotspot in number ofpixels from the left.
    bpp Number

    2 bytes. In ICO format: Specifies bits per pixel. In CUR format: Specifies the vertical coordinates of the hotspot in number of pixels from thetop.
    size Number

    4 bytes. Specifies the size of the image's data in bytes.
    offset Number

    4 bytes. Specifies the offset of BMP or PNG data from the beginning of the ICO/CUR file.

    See:

    • https://en.wikipedia.org/wiki/ICO_%28file_format%29

    public ICOHeader: Object source

    Header information of the ICO file.

    Properties:

    NameTypeAttributeDescription
    reserved Number

    2 bytes. Reserved. Must always be 0.
    type Number

    2 bytes. Specifies image type: 1 for icon (.ICO) image, 2 for cursor (.CUR) image. Other values are invalid.
    counr Number

    2 bytes. Specifies number of images in the file.
